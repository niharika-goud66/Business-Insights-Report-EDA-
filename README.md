# Task 04 - Business Insights Report (EDA)

## Project Title
Marketing Campaign Analysis Dashboard

## Objective
Perform Exploratory Data Analysis (EDA) on a marketing campaign dataset using Power BI to identify trends, evaluate campaign performance, and generate actionable business insights.

## Tools Used
- Power BI Desktop
- Microsoft Excel / CSV Dataset

## Project Description
This project focuses on analyzing marketing campaign data through interactive Power BI visualizations. The objective is to understand campaign effectiveness, customer engagement, and Return on Investment (ROI) to support better business decisions.

## Data Preparation
- Imported dataset into Power BI
- Checked for missing values
- Removed duplicate records
- Cleaned and transformed data
- Verified data consistency

## Dashboard Features
- KPI Cards
- Campaign Performance Analysis
- Marketing Channel Comparison
- ROI Analysis
- Customer Response Analysis
- Interactive Filters and Slicers
- Business Insights Dashboard

## Key Insights
- Identified the highest-performing marketing channels.
- Compared ROI across different campaigns.
- Analyzed customer engagement and conversion rates.
- Recommended channels for increased budget allocation.
- Highlighted opportunities for campaign improvement.

## Outcome
Successfully created an interactive Power BI dashboard and generated business insights to support data-driven marketing decisions.
